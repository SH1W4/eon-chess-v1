# XadrezMaster - Projeto Completo

## Estrutura do Projeto

```
/CHESS/
├── docs/              # Documentação
│   ├── arquitetura/   # Documentos de arquitetura
│   ├── analise/       # Análises e pesquisas
│   ├── produto/       # Documentação de produto
│   └── tecnico/       # Documentação técnica
│
├── src/               # Código fonte
│   ├── core/          # Lógica principal do jogo
│   ├── ui/            # Interface do usuário
│   └── utils/         # Utilitários
│
├── assets/            # Recursos
│   ├── images/        # Imagens e concept art
│   └── presentations/ # Apresentações
│
└── mobile/           # Aplicativo mobile
    ├── ios/          # Versão iOS
    └── android/      # Versão Android
```

## Descrição

XadrezMaster é um ecossistema completo que une xadrez tradicional com elementos inovadores de IA e aprendizado adaptativo.
