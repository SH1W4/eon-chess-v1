# AEON CHESS

Um ecossistema transcendente que une a sabedoria milenar do xadrez com IA adaptativa e aprendizado simbiótico.

## Estrutura do Projeto

```
/AEON
├── src/                # Código fonte
│   ├── core/          # Núcleo do sistema
│   ├── ai/           # Sistema de IA adaptativa
│   ├── mind/         # AEON MIND - Coaching IA
│   ├── ui/           # Interface do usuário
│   ├── cultural/     # Elementos culturais
│   └── utils/        # Utilitários
│
├── docs/               # Documentação
│   ├── technical/     # Documentação técnica
│   ├── product/      # Documentação de produto
│   ├── architecture/ # Arquitetura
│   └── analysis/     # Análises
│
├── assets/            # Recursos
│   ├── images/       # Imagens
│   │   ├── concept/  # Arte conceitual
│   │   ├── screens/  # Screenshots
│   │   └── icons/    # Ícones
│   ├── docs/        # Documentos
│   └── presentations/ # Apresentações
│
├── mobile/           # Aplicativo mobile
│   ├── ios/         # Versão iOS
│   └── android/     # Versão Android
│
└── config/          # Configurações
```

## Desenvolvimento

### Requisitos
- Node.js bz 18.0.0
- Python bz 3.11.0
- TypeScript bz 4.9.0

### Configuração
1. Clone o repositório
2. Instale as dependências:
   ```bash
   npm install     # Para dependências Node.js
   pip install -r requirements.txt  # Para dependências Python
   ```
3. Configure o ambiente:
   ```bash
   cp .env.example .env
   ```
4. Execute o projeto:
   ```bash
   npm start      # Inicia a interface web
   python src/ai/main.py  # Inicia o sistema de IA
   ```
