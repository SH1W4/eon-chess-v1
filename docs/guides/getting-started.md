# Guia de Início - AEON Chess

## Introdução

Bem-vindo ao AEON Chess, um sistema de xadrez inteligente que combina estratégia tradicional com IA adaptativa e aprendizado de máquina.

## Instalação Rápida

1. **Pré-requisitos**
   - Node.js ≥ 18.0.0
   - Python ≥ 3.11.0
   - Go ≥ 1.19.0
   - Docker & Docker Compose

2. **Configuração do Ambiente**
   ```bash
   # Clone o repositório
   git clone https://github.com/seu-usuario/aeon-chess.git
   cd aeon-chess

   # Configure o ambiente
   cp .env.example .env
   make setup
   ```

3. **Iniciando o Jogo**
   ```bash
   make up
   # Acesse http://localhost:3000
   ```

## Primeiros Passos

1. **Criando sua Conta**
   - Acesse a página inicial

2. **Tutorial Inicial**
