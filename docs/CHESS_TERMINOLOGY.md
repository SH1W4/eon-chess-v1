# Regras de Terminologia - CHESS

## Visão Geral

Este documento estabelece diretrizes para o uso consistente de terminologia no projeto CHESS, garantindo que os termos utilizados sejam apropriados ao contexto do xadrez e compreensíveis para os jogadores.

## Princípios Fundamentais

1. **Contextualização ao Xadrez**: Todos os termos devem ter relação direta ou metafórica com o universo do xadrez
2. **Clareza**: Preferir termos que sejam facilmente compreensíveis pelos jogadores
3. **Consistência**: Manter uso consistente dos termos em toda documentação
4. **Precisão Técnica**: Usar terminologia técnica apenas quando necessário e com explicação adequada

## Substituições Terminológicas

### Termos a Evitar vs. Termos Preferidos

| Termo a Evitar | Termo Preferido | Contexto de Uso |
|----------------|-----------------|-----------------|
| Simbiótico | Adaptativo | Para descrever interação sistema-jogador |
| Consciência | Análise | Para capacidades do sistema |
| Transcendente | Avançado | Para níveis de habilidade |
| Dimensional | Posicional | Para aspectos do jogo |
| Quântico | Dinâmico | Para comportamentos do sistema |

### Contexto Cultural

| Termo a Evitar | Termo Preferido | Contexto de Uso |
|----------------|-----------------|-----------------|
| Ritual | Estratégia | Para práticas específicas |
| Ancestral | Tradicional | Para técnicas históricas |
| Sagrado | Clássico | Para posições/jogadas importantes |
| Iluminação | Maestria | Para níveis de domínio |

## Frameworks e Sistemas

### Sistema AEON MIND
- Usar "Sistema de Análise Adaptativa" ou "Motor de Coaching"
- Focar em termos relacionados a treinamento e desenvolvimento
- Evitar referências místicas ou esotéricas

### Elementos Culturais
- Contextualizar sempre à história do xadrez
- Usar termos específicos de cada escola/tradição de xadrez
- Relacionar a grandes mestres e momentos históricos

## Níveis de Desenvolvimento

| Nível | Descrição Contextualizada |
|-------|---------------------------|
| Iniciante | Aprendendo movimentos básicos |
| Intermediário | Desenvolvendo estratégias |
| Avançado | Dominando táticas complexas |
| Mestre | Criando estratégias inovadoras |

## Aspectos Técnicos

### IA e Aprendizado de Máquina
- Usar "análise posicional" em vez de "análise neural"
- Preferir "adaptação dinâmica" a "evolução simbiótica"
- Descrever capacidades em termos de xadrez (ex: "avaliação posicional", "análise tática")

### Interface e Interação
- Relacionar comandos a movimentos do xadrez
- Usar analogias com elementos do tabuleiro
- Manter referências visuais alinhadas com a tradição do xadrez

## Exemplos de Uso

### Correto
- "O sistema adapta seu nível baseado em suas partidas"
- "Análise profunda de posições críticas"
- "Desenvolvimento progressivo de habilidades"
- "Motor de avaliação dinâmica"
