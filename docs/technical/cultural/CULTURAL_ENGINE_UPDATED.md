## Exemplos de Uso

```python
# Inicialização do Motor Cultural
from aeon_chess.cultural import CulturalEngine

engine = CulturalEngine(
    config_path="/config/cultural_engine.yaml",
    cache_enabled=True,
    adaptive_mode=True
)

# Processamento de Contexto Cultural
context = engine.process_cultural_context({
    "historical_period": "classical",
    "region": "eastern_europe",
    "style": "positional"
})

# Geração de Narrativa Cultural
narrative = engine.generate_cultural_narrative(
    context=context,
    player_profile=player,
    game_state=current_game
)

# Evolução de Padrões Culturais
engine.evolve_cultural_patterns(
    feedback=user_feedback,
    performance_metrics=metrics,
    adaptation_rate=0.1
)
```
