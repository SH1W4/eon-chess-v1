## Exemplos

```python
from aeon_chess.cultural import CulturalEngine

# Inicialização
engine = CulturalEngine(
    config="cultural.yaml",
    cache=True,
    adaptive=True
)

# Contexto
ctx = engine.process_context({
    "period": "classical",
    "region": "europe",
    "style": "tactical"
})

# Narrativa
story = engine.generate_narrative(
    context=ctx,
    profile=player,
    game=current
)

# Evolução
engine.evolve(
    feedback=user_input,
    metrics=stats,
    rate=0.1
)
```
